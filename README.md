

# Cypress End-to-End Testing Project

## Description
This project demonstrates end-to-end (E2E) testing using the Cypress framework. The tests are designed to validate different login scenarios for the website [https://the-internet.herokuapp.com/](https://the-internet.herokuapp.com/).

## Test Cases
1. Valid password and invalid username.
2. Invalid password and invalid username.
3. Valid username and valid password.
4. Other edge cases.

## Steps to Run the Project
1. Clone this repository.
2. Install Node.js and Cypress.
3. Run the Cypress tests using the following command:
    ```bash
    npm install cypress
    npx cypress open
    ```
